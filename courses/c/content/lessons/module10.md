## File I/O

### Topics Covered

- Opening and closing files with fopen/fclose
- Reading: fgetc, fgets, fscanf, fread
- Writing: fputc, fputs, fprintf, fwrite
- Binary vs text mode
- File positioning: fseek, ftell, rewind
- Error handling with errno and perror
- stdin, stdout, stderr
