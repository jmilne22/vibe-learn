## Multithreaded Web Server

> Based on The Rust Programming Language, Chapter 20

### Project:

- Build a multithreaded web server with a thread pool
- Listen for TCP connections
- Parse HTTP requests
- Write HTTP responses
- Implement a thread pool for concurrent request handling
- Graceful shutdown and cleanup
