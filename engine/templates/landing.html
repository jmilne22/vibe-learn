// vibe-learn

Build an interactive course
on anything

Most courses teach a language in the abstract — syntax, toy exercises, nothing real. Create a course built around your goal: Go for CLI tools, Rust for a game engine, Python for your data pipeline.

Tell Claude Code, Cursor, Copilot, or Augment what to teach. One build command gives you exercises, spaced repetition, flashcards, and analytics.

// zero config

Content files drive features — no registries, no config files, no wiring. Drop files in the right place and the build figures out the rest.

>
Drop a course folder → it appears on the site
The build scans courses/ for any directory with a course.yaml. Add a folder, rebuild, done.
courses/python/ + course.yaml → auto-discovered
>
Add a content file → the feature activates
Each plugin declares a content pattern. If a course has matching files, it gets the feature. No files? The page doesn't exist — no dead links.
content/flashcards/flashcards.yaml exists → flashcards page active
>
Visual course builder (WIP)
Prefer not to use AI? npm run builder launches a local web editor with forms for every content type, live markdown preview, and one-click builds.
npm run builder → http://localhost:3456
